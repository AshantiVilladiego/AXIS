# AXIS
A.X.I.S. (Automated eXtraction &amp; Integration System) 

This project is a web‑based SaaS that uses LLM integration to let users upload messy government forms and automatically generate clean, pre‑filled PDFs or step‑by‑step compliance guides.

## Tech Stack
* **Frontend:** Next.js (React)
* **Backend:** FastAPI (Python)
* **Database:** PostgreSQL
* **CI/CD & Deployment:** Vercel (Frontend)
